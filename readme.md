## 🧮 Scientific Calculator

### 🧾 Overview
A feature-rich scientific calculator built with JavaFX. This application supports basic arithmetic, advanced mathematical operations, trigonometric functions, calculus computations, and equation solving.

It is designed with a clean and interactive user interface, making it suitable for students, engineers, and anyone needing powerful mathematical tools in a desktop environment.

---

### 🚀 Features

#### 🔢 Basic Operations
- Addition, subtraction, multiplication, division  
- Parentheses support for complex expressions  
- Decimal input and sign change  

#### 💾 Memory Functions
- MC (Memory Clear)  
- MR (Memory Recall)  
- MS (Memory Store)  
- M+ (Memory Add)  
- M− (Memory Subtract)  

#### 📐 Trigonometric Functions
- sin, cos, tan, cot, sec, csc  
- Inverse functions: asin, acos, atan  
- Degree and Radian modes  

#### 📊 Logarithmic Functions
- Natural logarithm (ln)  
- Logarithm with custom base  

#### 🧠 Advanced Mathematical Operations
- Square (x²)  
- Square root (√x)  
- Absolute value (|x|)  
- Factorial (n!)  
- Constants: π and e  

#### 📉 Calculus Features
- Derivative calculation  
- Numerical integration (Simpson’s Rule)  
- Limit approximation  

#### 🧮 Equation Solvers
- Linear equations: ax + b = 0  
- Quadratic equations: ax² + bx + c = 0  

#### 📌 Additional Features
- Variable `x` support for expressions  
- Calculation history (stores up to 50 entries)  
- Standard and Advanced modes  

---

### 🛠️ Tech Stack
- **Java**  
- **JavaFX** (UI framework)  
- **FXML** (UI structure)  
- **CSS** (styling)  

---

### ⚙️ Project Structure

- `src/calculator/CalculatorApp.java` — Application entry point. Initializes JavaFX and loads the FXML UI.
- `src/calculator/UI/calculator.fxml` — The UI layout for the calculator (FXML). Students should continue to use FXML for all UI changes.
- `src/calculator/Controller/CalculatorController.java` — Controller class; connects UI elements declared in the FXML with application logic.
- `src/calculator/styles/styles.css` — Stylesheet for simple styling.
- `build/` — Compiled classes and build artifacts (ignored when contributing).
